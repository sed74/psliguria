# Pronto Soccorso Liguria
Questa app ha le seguenti funzionalità:
 * Permette di vedere le code di attesa presso i Pronto Soccorsi genovesi, col numero di pazienti sia in visita che in attesa di visita per ogni codice di gravità.
 * Permette di visualizzare, in tempo reale, le ospedalizzazioni ad opera di CRI e di Pubbliche Assistenze su tutto il territorio ligure. Per ogni missione vengono visualizzati il luogo di intervento, la tipologia di patologia, il codice di urgenza e l'ospedale di destionazione oltre, ovviamente al nome della pubblica assistenza e al numero dell'ambulanza.
